# 💰 ZenLedger — Decentralized E2EE Personal Finance & Expense Tracker

ZenLedger is a lightweight, zero-knowledge **Personal Finance, Income, and Expense Tracker** running serverless inside the browser. It encrypts all transaction records, budgets, and balance metrics locally using WebCrypto API before syncing them across devices via `@akaoio/zen`.

Powered by **ZEN**—a zero-config decentralized peer-to-peer graph database.

## 🚀 Features

- 💵 **Income & Expense Logging**: Track transactions with amount, category, date, payment type, and notes.
- 📊 **Financial Dashboard**: Real-time Net Balance, Total Income, Total Expenses, and Category Breakdown progress metrics.
- 🔐 **End-to-End Encrypted**: Zero-knowledge financial data stored securely in your private graph namespace.
- ⚡ **Real-Time P2P Sync**: Seamless sync across laptop and mobile devices.
- 🎨 **Themeable**: Light & Dark themes, Outfit & IBM Plex Mono typography.
